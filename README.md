# 🎮 Resident Evil Forum

Aplicação fullstack inspirada na franquia Resident Evil, onde usuários podem visualizar jogos da saga, comentar e avaliar conteúdos cadastrados por administradores.

---

## 🚀 Tecnologias

- Backend: Node.js + Express + TypeScript
- Banco de dados: PostgreSQL (Neon) + Prisma ORM
- Frontend: React + Vite

---

## 🎯 Objetivo

Desenvolver uma aplicação completa com:

- API REST
- Integração entre backend e frontend
- Modelagem de banco de dados relacional
- Boas práticas de organização de projeto

---

## 📌 Funcionalidades

- Cadastro de jogos (admin)
- Listagem de jogos
- Cadastro de usuários
- Avaliações com comentários
- Exibição de imagens dos jogos

---

## ⚙️ Como rodar o projeto

## Backend
npm i / npm run dev

## Frontend
npm i / npm run dev
